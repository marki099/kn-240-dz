---### Основи Python (коротко)

1. **Що таке Python?**  
   Легка у вивченні мова програмування, використовується для веброзробки, аналізу даних, автоматизації та ІІ.

2. **Змінні та типи даних**  
   - `int`, `float`, `str`, `bool`.  
   - Приклад:  
     ```python
     x = 10
     name = "Python"
     is_active = True
     ```

3. **Умови**  
   ```python
   if x > 5:
       print("x більше 5")
   else:
       print("x менше або дорівнює 5")
   ```

4. **Цикли**  
   - `for` і `while`:  
     ```python
     for i in range(3):
         print(i)
     ```

5. **Функції**  
   ```python
   def greet(name):
       return f"Привіт, {name}!"
   ```

6. **Списки та словники**  
   - Список: `fruits = ["яблуко", "банан"]`  
   - Словник: `person = {"ім'я": "Аня", "вік": 20}`

7. **Імпорт модулів**  
   ```python
   import math
   print(math.sqrt(16))
   ```
